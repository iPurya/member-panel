# member-panel
Member panel that is a bot on telegram, for add channel/group members.



## Install :

apt-get -y update && apt-get -y install git  
git clone https://github.com/iPurya/member-panel.git  
cd member-panel  
chmod +x *  
./install.sh


## Run :

tmux  
./bot
